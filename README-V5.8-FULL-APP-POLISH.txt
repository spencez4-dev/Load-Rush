# Load Rush V5.8 — Full App Polish

A light, app-wide polish pass built on V5.7 Dispatch Analytics.

## Changes
- Stronger visual hierarchy around the main counter and primary + button
- Smaller, quieter minus control
- Micro XP float, truck suspension bounce, road pulse, and short trail on updates
- Activity tools simplified: Analytics and Day Recap stay visible; Undo and Export move into a more menu
- Freight Fate now rolls directly on the road and auto-clears without blocking dialogs
- Unified naming: Beat the Clock, race wins, Analytics, and 15-load race language
- More consistent dialog spacing, corners, shadows, and mobile sizing
- Cleaner empty states, subtler build badge, and lighter secondary information
- Reduced-motion support for accessibility
- Existing saved data remains compatible

LOAD RUSH V3.1 — REMINDERS UPDATE

WHAT WAS ADDED
- Reminders button in the top navigation
- User-entered reminder message
- Date and time inputs
- Saved reminder list
- Delete reminders before they fire
- Full-screen flashing alarm
- Repeating loud siren
- Vibration on supported devices
- Browser/system notification when permission is granted
- Large I GOT IT dismissal button

IMPORTANT BROWSER BEHAVIOR
The full-screen alarm, siren, and reminder checker work while the app is open.
System notifications can appear while the tab is in the background when notification
permission is granted. A static GitHub Pages app cannot guarantee an alarm after the
browser/app has been fully closed.
